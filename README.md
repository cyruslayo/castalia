## Why this layout

The roadmap is intentionally holistic rather than linear:

- evaluate before and while you build
- revisit prompting, retrieval, and tool design at increasing depth
- treat finetuning as a later lever, not the first move
- connect multimodal, agents, and systems to the same measurement mindset

If you want the most complete view, start with [ROADMAP.md](ROADMAP.md).
